  2. List types

    The slides support several list types:

    Definition lists
    This list is a perfect example for a definition list.
    Ordered / unordered lists
    Regular lists, but by default there will be no bullet points for top-level unordered lists.
    Static lists
    When in interactive mode (see next slide), items of unordered lists will be displayed step-wise. Disable this behaviour by adding the CSS class static to the <ul> element.

  3. Unordered lists

    This is a second demo slide including an unordered list. Press SHIFT to toggle the interactive mode for lists.

    • The first three list items are static
    • and won't be displayed step-wise
    • regardless whether you're in interactive mode or not.
    • The last three items, however,
    • will be show item by item
    • when you're in interactive mode and press the forward arrow key.
