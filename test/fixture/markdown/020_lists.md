## List types

The slides support several list types:

<dl>
    <dt>Definition lists</dt>
    <dd>This list is a perfect example for a definition list.</dd>
    <dt>Ordered / unordered lists</dt>
    <dd>Regular lists, but by default there will be no bullet points for top-level unordered lists.</dd>
    <dt>Static lists</dt>
    <dd>When in interactive mode (see next slide), items of unordered lists will be displayed step-wise. Disable this behaviour by adding the CSS class <code>static</code> to the <code>&lt;ul&gt;</code> element.</dd>
</dl>
